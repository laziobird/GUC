GUC: 通用用户中心系统

* GUC: 简称General User Center，它是基于SpringCloud实现的一种微服务

* GUC为不同客户提供了系统授权、用户认证UAA一整套服务解决方案

* UAA: User Account and Authentication

* GUC是一个独立的微服务，可以方便在私有云和公有云环境部署。它丰富的RESTFul API可以在IT系统、移动APP程序方便调用、扩展
